# Dev Suthar - Portfolio

Welcome to the source code of my personal developer portfolio! This project is a highly interactive, beautifully animated, and feature-rich portfolio website built to showcase my skills, projects, and experience as a software developer.

## ✨ Features

- **Modern UI/UX**: Premium design aesthetics using glassmorphism, dynamic gradients, and a Bento Grid layout.
- **Smooth Animations**: High-performance scrolling and micro-interactions powered by Framer Motion.
- **Dynamic Projects Showcase**: Stacked scrolling cards with deep-link support and individual project detail pages.
- **Dark/Light Mode**: Full theme support with a custom-designed toggle switch.
- **Firebase Integration**: Real-time database for access requests, logging, and security tracking.
- **Fully Responsive**: Optimized for desktops, tablets, and mobile devices with custom scroll behaviors.

## 🛠️ Tech Stack

- **Frontend Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS & Vanilla CSS (Custom tokens)
- **Animations**: Framer Motion
- **Database & Auth**: Firebase (Firestore & Google Authentication)
- **Routing**: React Router DOM
- **Icons**: Lucide React & Devicon


## 📂 Project Structure

- `/src/components` - Reusable UI components and specific section files (Hero, About, Projects, etc.)
- `/src/pages` - Main page routes (Index, ProjectDetail, AdminDashboard, NotFound)
- `/src/data` - Local JSON data files (e.g., `projects.ts` containing all project details)
- `/src/lib` - Utility configurations (Firebase initialization, etc.)
- `/public` - Static assets and images
